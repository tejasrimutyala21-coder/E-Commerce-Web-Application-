Elite Mart E-Commerce Web Application

Project Overview

Elite Mart is a simple E-Commerce Web Application developed using HTML, CSS, and JavaScript. The website allows users to browse products, search for products, add items to a shopping cart, and place orders through a checkout system.

Features

- Product Catalog Display
- Product Search Functionality
- Add to Cart Feature
- Shopping Cart Management
- Total Price Calculation
- Checkout System
- Responsive User Interface
- Services Section
- Navigation Menu

Technologies Used

- HTML5
- CSS3
- JavaScript

Project Structure

- HTML: Website structure and content
- CSS: Styling and responsive design
- JavaScript: Cart management, search functionality, and checkout process

Available Products

1. Laptop
2. Smartphone
3. Wireless Earbuds
4. Smart Watch
5. Men's Shoes
6. Women's Handbag
7. Bluetooth Speaker
8. Gaming Mouse

How to Run the Project

1. Install Visual Studio Code.
2. Create a file named "index.html".
3. Copy the source code into the file.
4. Save the file.
5. Open the file in any web browser.
6. Start using the application.

Functionalities

Product Search

Users can search products using the search bar.

Add to Cart

Users can add products to the shopping cart by clicking the "Add to Cart" button.

Shopping Cart

The cart displays selected products and calculates the total amount automatically.

Checkout

Users can place orders through the checkout button. After checkout, the cart is cleared and the order is confirmed.
